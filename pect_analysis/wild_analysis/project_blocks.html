project_blocks Table

Overview

Each row in this table represents a project shared on the Scratch website at the point of data collection. Each column of this row represents the frequency counts of each block type for a particular project. The value of each cell represents the number of times a particular block is used in a particular project.

Projects use visual programming blocks to determine what projects do. Each block was stored internally as a token and each block could be used zero or more times.

Description of Fields

This table includes 1,925,606 observations. Each row of this table corresponds to a single project.

Blocks were organized in eight visible categories that included "looks" (i.e. blocks that let users change the look of a sprite), "sensing" (e.g. blocks used to detect colors and mouse position), "operators' (e.g. blocks for arithmetic operations).

The description column tries to match the way the block was rendered in the Scratch user interface. Blocks that end with "_" are blocks that take an argument. Those without a description are experimental or obsolete blocks.

Omitted Observations

We omit 487,528 observations because they refer to observations in older, non-public, versions of projects that were replaced on the site. Additionally, we omit 498,074 observations from projects which are not included in our projects table (e.g., they were removed from the site by their creators). See the projects table for details on these omitted data.
